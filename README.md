### React Solitaire

https://sdoc-games.vercel.app

Welcome to my own personal version of Klondike solitaire.
This is a work of progress, with more card games (outside of solitaire) to come. Hope you enjoy! This version of solitaire is done entirely in react, with the help of framer motion to visualize card movements. No game engine or framework is used, with game logic being implemented mainly using state and context variables. 
Since this project does not use any sort of canvas or engine, you'll find that its slightly smoother performance-wise than other card games. No images are being rendered, as every element, down to the cards & their designs are entirely done within html components using CSS.
